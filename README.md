# Express pattern MVC - Simple App product details

1. Create files:
    - **models**.
    - **controllers**.
    - routes.
    - **views**.
    - public (css, js, etc).
    - *data*.
2. settings (create `index.js` or `app.js`).
    - Initialize express app.
    - Settings of express -> `app.set()`
    - Middlewares -> `app.use()`
    - Routes -> `app.use()`
    - Serverlisten -> `app.listen()`
